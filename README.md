## 👋 About SIAutomater-safari

SIAutomater-safari adalah safari extension untuk melakukan pengisian kuisioner secara otomatis pada laman SIA UNSOED. Tool ini merupakan solusi bagi orang yang mager seperti creatornya. Jadi, harap digunakan dengan bijak dan jangan gegabah.

Kalau kamu punya kritik/saran mengenai tool ini ataupun kamu dapet error yang gajelas, bisa kontak lewat instagram [@falll.hkm](https://instagram.com/fall.hkm)👋😊 <br><br>

## ⚙️ Installation

1. Buka laman [GitHub SIAutomater](https://github.com/ibrahimirfn/siautomater-safari)

   <img src="/images/tutorial/installation/1.png" alt="Laman SIA" width="500"><br><br>

2. Download source codenya dengan cara pilih ***Code*** lalu klik ***Download ZIP***

   <img src="/images/tutorial/installation/2.png" alt="Laman SIA" width="500"><br><br>

3. Extrak file zip yang sudah didownload

   <img src="/images/tutorial/installation/3.png" alt="Laman SIA" width="300"><br><br>

4. Pada Safari, klik ***setting-advanced*** dan pilih ***show features for web developers***

   <img src="/images/tutorial/installation/4.png" alt="Laman SIA" width="500"><br><br>

5. Aktifkan ***unsigned extensions*** dan klik ***add temporary extensions***

   <img src="/images/tutorial/installation/5.png" alt="Laman SIA" width="500"><br><br>

6. Cari folder SIAutomater yang sudah di ekstrak tadi dan klik ***select***

   <img src="/images/tutorial/installation/6.png" alt="Laman SIA" width="500"><br><br>

7. Cari SIAutomater di Extensions, jika sudah ada selamat instalasi berhasil

   <img src="/images/tutorial/installation/7.png" alt="Laman SIA" width="500">

## 🚀 Usage

1. Buka laman [SIA 5.0](https://sia.akademik.unsoed.ac.id/auth/login)

   <img src="/images/tutorial/usage/1.png" alt="Laman SIA" width="500"><br><br>

2. Klik login dan lakukan login menggunakan akun kamu

   <img src="/images/tutorial/usage/2.png" alt="Laman SIA" width="500"><br><br>

3. Jika berhasil maka kamu akan diarahkan ke halaman dashboard

   <img src="/images/tutorial/usage/3.png" alt="Laman SIA" width="500"><br><br>

4. Klik extension SIAutomater yang sudah di install

   <img src="/images/tutorial/usage/4.png" alt="Laman SIA" width="300"><br><br>

5. Isi keterangan yang dibutuhkan pada extension sesuai hati nurani mu

   <img src="/images/tutorial/usage/5.png" alt="Laman SIA" width="300"><br><br>

6. Klik tombol "run" 🚀

   <img src="/images/tutorial/usage/6.png" alt="Laman SIA" width="300"><br><br>

7. Tunggu hingga proses selesai ✨ <br><br>
