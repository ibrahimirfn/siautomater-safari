## 👋 About SIAutomater

SIAutomater adalah chrome extension untuk melakukan pengisian kuisioner secara otomatis pada laman SIA UNSOED. Tool ini merupakan solusi bagi orang yang mager seperti creatornya. Jadi, harap digunakan dengan bijak dan jangan gegabah.

Kalau kamu punya kritik/saran mengenai tool ini ataupun kamu dapet error yang gajelas, bisa kontak lewat instagram [@falll.hkm](https://instagram.com/fall.hkm)👋😊

## ⚙️ Installation

1. Buka laman [GitHub SIAutomater](https://github.com/kimnopal/siautomater) <br><br>
   <img src="/images/tutorial/installation/1.png" alt="Laman SIA" width="500"> <br>
2. Download source codenya dengan cara pilih **Code** lalu klik **Download ZIP** <br><br>
   <img src="/images/tutorial/installation/2.png" alt="Laman SIA" width="500"><br>
3. Extrak file zip yang sudah didownload <br><br>
   <img src="/images/tutorial/installation/3.png" alt="Laman SIA" width="300"><br>
4. Pada Chrome, klik **icon extension** dan pilih **Manage extensions** <br><br>
   <img src="/images/tutorial/installation/4.png" alt="Laman SIA" width="500"><br>
5. Aktifkan **Developer mode** dengan mengeklik tombol togglenya <br><br>
   <img src="/images/tutorial/installation/5.png" alt="Laman SIA" width="500"><br>
6. Pilih **Load unpacked** <br><br>
   <img src="/images/tutorial/installation/6.png" alt="Laman SIA" width="500"><br>
7. Cari folder SIAutomater yang sudah di ekstrak tadi dan klik **select folder** <br><br>
   <img src="/images/tutorial/installation/7.png" alt="Laman SIA" width="500"><br>
   **Note:**
   Perhatikan dalam memilih folder, karena biasanya folder hasil diekstrak memiliki sub folder lagi.

   Jika kamu mengalami error seperti ini, maka kamu salah memilih folder <br><br>
   <img src="/images/tutorial/installation/8.png" alt="Laman SIA" width="500"><br>

8. Selesai, kamu bisa langsung menggunakan SIAutomater untuk mengatasi ke-mageranmu dalam mengisi kuisioner😁 <br><br>

## 🚀 Usage

1. Buka laman [SIA 5.0](https://sia.akademik.unsoed.ac.id/auth/login) <br><br>
   <img src="/images/tutorial/usage/1.png" alt="Laman SIA" width="500"><br>
2. Klik login dan lakukan login menggunakan akun kamu <br><br>
   <img src="/images/tutorial/usage/2.png" alt="Laman SIA" width="500"><br>
3. Jika berhasil maka kamu akan diarahkan ke halaman dashboard <br><br>
   <img src="/images/tutorial/usage/3.png" alt="Laman SIA" width="500"><br>
4. Klik extension SIAutomater yang sudah di install <br><br>
   <img src="/images/tutorial/usage/4.png" alt="Laman SIA" width="300"><br>
5. Isi keterangan yang dibutuhkan pada extension sesuai hati nurani mu <br><br>
   <img src="/images/tutorial/usage/5.png" alt="Laman SIA" width="300"><br>
6. Klik tombol "run" 🚀 <br><br>
   <img src="/images/tutorial/usage/6.png" alt="Laman SIA" width="300"><br>
7. Tunggu hingga proses selesai ✨ <br><br>
